# react
Use react components in the Vue project

```
yarn add --save vext-react
```

```
npm i --save vext-react
```

``` ts
import VueReact from "vext-react";
Vue.use(VueReact);

```

Use in the vue file:

``` ts
import TweenOne from "rc-tween-one";
Vue.useReact("TweenOne", TweenOne);
```
